Frequently Asked Questions

Is this official?

This is a community project to simplify Claude Code installation. It installs the official Claude Code package from npm.

Is it safe?

Yes! The code is open-source and available on GitHub. It only installs official packages (Node.js and Claude Code). No malware, no tracking.

Do I need administrator rights?

No for most cases. The installer uses per-user installation. Admin rights may help but aren't required.

What if I already have Node.js?

The installer detects existing Node.js. If version 18+ is installed, it skips Node.js installation and only installs Claude Code.

How do I uninstall?

Windows: Settings ‚Üí Apps ‚Üí Node.js/Claude Code. macOS: npm uninstall -g @anthropic-ai/claude-code then brew uninstall node@20

Where can I get help?

Check the Quick Start Guide or open an issue on GitHub.
